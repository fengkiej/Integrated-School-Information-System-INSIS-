Instalasi INSIS - Step I

Hai! Selamat datang di INSIS (Integrated School Information System), sebuah platform sistem informasi open-source lengkap yang sangat mudah untuk digunakan.

Sebelum memulai, Anda harus melakukan beberapa konfigurasi untuk sistem ini. Tenang saja, ini hanya akan memakan waktu beberapa menit. Kami akan menuntun Anda step-by-step untuk melakukannya (tidak perlu pengetahuan teknis yang rumit untuk memulai!)

Ada beberapa hal teknis yang harus Anda siapkan sebelum melakukan konfigurasi.

  • Nama database
  • Username database
  • Password database
  • Nama host database
  • Username administrator
  • Password administrator

Dan akan ada beberapa detail non-teknis yang harus diisi untuk memulai.

  • Nama sekolah
  • NPSN
  • Tingkat pendidkan (SD/SMP/SMA/SMK)
  • Alamat sekolah
    • Jalan
    • Kode pos
  • Website sekolah
  • Nomor telp.
  • Nomor fax.
  • Alamat email sekolah

Jika Anda masih merasa cara kebingungan untuk memulai, silahkan akses tutorial instalasi melalui http://insis.id/tutorial. Jika Anda sudah siap, ayo klik tombol 'Lanjut!' :)

Lanjut!
